About This Repository
----------------------

This monorepo contains all Boomchecker components:

* **Firmware**: ESP-IDF-based firmware for embedded devices
* **Node.js Apps**: Backend services and APIs
* **Scripts**: Supporting tools and automation scripts

Each documentation area is independently versioned and maintained.
